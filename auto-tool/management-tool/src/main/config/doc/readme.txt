PrimeCloud Controller ユーザ管理ツール 概要 ver 1.4.1
最終更新日 2011/3/10

本製品はPrimeCloud Controller(以下PCC)のユーザを管理するためのソフトウェアです。
コマンドを実行することにより、各種操作をすることが出来ます。

■ドキュメント構成

　□readme.txt : ドキュメントの概要
　　各ドキュメントの概要を記述しています。

　□install-manual.txt : インストールマニュアル
　　本製品をインストールする為の手順を記述しています。

　□usecase.txt : ユースケースマニュアル
　　作業における操作ケースごとに使用するコマンドを記述しています。
　　
　□reference.txt : コマンドリファレンス
　　コマンドごとの詳細を記述しています。

■使用可能な操作

　ユーザ作成
　ユーザ一覧表示
　プラットフォーム一覧表示
　AWSプラットフォーム有効化
　VMwareプラットフォーム有効化
　CloudStackプラットフォーム有効化
　ユーザパスワード変更
　ユーザ無効化
　ユーザ有効化
　ヘルプ表示

■使用不可能な操作

　ユーザ削除
　ユーザのプラットフォームの無効化
　キーペアの変更
　AWSの認証情報の変更
　CloudStackの認証情報の変更
　Eucalyptus、NiftyCloudのプラットフォーム有効化
